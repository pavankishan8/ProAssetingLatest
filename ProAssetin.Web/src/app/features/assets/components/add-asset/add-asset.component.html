❓ Excel Format Instructions

Required Columns:

  • AssetId - Unique asset identifier (required)
  • Name - Asset name (required)
  • Category - Asset category (required)
  • Status - In-Stock, Repair, Sold, Damaged, or E-Waste (required)
  • Location - Asset location (optional)
  • Make - Manufacturer (optional)
  • Model - Model number (optional)
  • SerialNumber - Serial number (optional)
  • PurchasePrice - Purchase price (optional)
  • PurchaseDate - Purchase date in MM/DD/YYYY format (optional)
  • WarrantyExpiryDate - Warranty expiry in MM/DD/YYYY format (optional)
  • Description - Additional notes (optional)

Note: First row should contain column headers. Data starts from row 2.
